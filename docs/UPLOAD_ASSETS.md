# Upload Assets

在上传到 QGIS 插件仓库前，建议准备以下素材：

1. `docs/screenshots/dock-panel.png`
主聊天停靠面板，展示发送命令、计划说明和执行结果。

2. `docs/screenshots/settings-dialog.png`
设置页，展示 DeepSeek / Gemini / OpenAI Compatible 等 provider 配置。

3. `docs/screenshots/standard-rendering.png`
图层按规程标准渲染的结果图，最好能体现规划图层或控制线配色。

4. `resources/icon.png`
插件图标，当前已提供 PNG 版本。

建议：

- 截图尺寸尽量统一，宽度控制在 1400-1800 像素。
- 截图中尽量保留 QGIS 图层树、地图画布和插件面板，避免只截弹窗。
- 使用真实业务数据前先脱敏。
