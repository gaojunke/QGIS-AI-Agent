# Release Notes

## v0.3.0

发布日期：2026-03-26

### 概述

`v0.3.0` 是一次面向发布体验和入口可用性的收口版本，重点修复设置入口不明显的问题，并补齐更适合公开展示和上传发布的文档与打包流程。

### 本次更新

- 恢复 API 设置入口
- 在 `Plugins` 菜单重新加入插件与设置项
- 保留 `Vector` 菜单入口
- 在主面板顶部增加 `API设置` 按钮
- 优化 GitHub 首页 README 展示
- 补充正式 Release Notes
- 生成可直接上传的干净 ZIP 包

## v0.2.0

发布日期：2026-03-25

### 概述

`v0.2.0` 是 `QGIS AI Agent` 的首个完整发布版本，重点围绕规划业务场景把插件从“原型可用”推进到“可打包、可上传、可演示”的状态。

### 主要新增

- 新增聊天式 QGIS 智能交互面板
- 新增工程上下文读取、短会话记忆与最近操作记录
- 新增本地图层问答能力
- 新增常见 Processing 算法执行与多步链式规划
- 新增图层常见操作入口，如缩放、显示隐藏、属性表、布局管理等
- 新增图层单色渲染、分类渲染与按规程标准渲染
- 新增标准样式库机制，支持附录 B/C/D 等规划制图标准配色

### 模型支持

- 支持 DeepSeek
- 支持 Gemini
- 支持 OpenAI Compatible API
- 支持 Ollama
- 支持托管后端模式

### DeepSeek 升级

- 增加独立 DeepSeek provider 实现
- 支持 `deepseek-chat` 和 `deepseek-reasoner`
- 支持 `thinking` 参数开关
- 支持 tool calling 规划模式
- 支持在面板中展示 reasoning 内容

### 工程化改进

- 增加执行取消机制和状态更新
- 优化上下文缓存，减少重复扫描工程
- 增强错误提示与回退逻辑
- 改进设置页异步网络请求体验
- 清理打包结构，补齐插件元数据
- 增加一键发布打包脚本

### 上传准备

- 插件名称统一为 `QGIS AI Agent`
- 元数据修正为适合 QGIS 插件仓库上传的格式
- 增加 PNG 图标
- 增加 GitHub 仓库、主页和问题跟踪链接
- 生成标准 ZIP 发布包

### 已知限制

- 当前仍以结构化计划执行为主，不支持任意 Python 自动执行
- 真实截图素材仍需补充，用于 QGIS 插件仓库展示
- 复杂行业编码映射和更多规程样式仍可继续扩展
